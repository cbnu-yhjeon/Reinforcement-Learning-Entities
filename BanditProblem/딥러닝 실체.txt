퍼셉트론(Perceptron)은 프랭크 로젠블랫(Frank Rosenblatt)이 1958년에 제안한 초기 인공 신경망 모델입니다. 퍼셉트론은 생물학적 뉴런의 동작 방식을 모방하여 설계되었으며, 단순한 이진 분류 문제를 해결하는 데 사용됩니다딥러닝의 선구자, 로젠블랫](https://www.letr.ai/blog/story-20211119-1).

퍼셉트론의 기본 구조는 다음과 같습니다:
1. **입력층(Input Layer)**: 여러 개의 입력 노드로 구성되며, 각 입력 노드는 하나의 특성 값을 나타냅니다.
2. **가중치(Weights)**: 각 입력 노드에 연결된 가중치는 입력 값의 중요도를 나타내며, 학습 과정을 통해 조정됩니다.
3. **활성화 함수(Activation Function)**: 입력 값과 가중치의 선형 결합 결과를 비선형적으로 변환하여 출력 값을 결정합니다.
4. **출력층(Output Layer)**: 최종 예측 결과를 나타내는 단일 노드로 구성됩니다.

퍼셉트론은 입력 값과 가중치의 선형 결합 결과가 특정 임계값을 초과하면 1을 출력하고, 그렇지 않으면 -1을 출력합니다. 이를 통해 이진 분류 문제를 해결할 수 있습니다.

퍼셉트론은 초기 인공 신경망 연구의 중요한 기초를 마련했지만, XOR 문제와 같은 비선형 분류 문제를 해결하지 못하는 한계가 있었습니다. 이후 다층 퍼셉트론(Multilayer Perceptron)과 역전파 알고리즘(Backpropagation)이 개발되면서 이러한 한계를 극복하게 되었습니다.
